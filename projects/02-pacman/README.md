# Projeto 02 — Pac-Man (Console) Lite

## Como compilar (exemplo)
```bash
g++ src/main.cpp -O2 -std=c++17 -o pacman.exe
```

## Como rodar
```bash
./pacman.exe
```

## Requisitos mínimos
- Mapa em matriz 2D
- Controle com WASD
- Fantasmas com movimento simples
- Pontuação e vidas

> Após rodar, tire um print e envie no tópico do projeto para liberar o próximo módulo.
