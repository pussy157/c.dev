# Projeto 01 — Jogo da Velha (Console)

## Como compilar (exemplo)
Se você estiver no Windows com MinGW (g++):
```bash
g++ src/main.cpp -O2 -std=c++17 -o jogo_da_velha.exe
```

## Como rodar
```bash
./jogo_da_velha.exe
```

## Regras
- 2 jogadores (X e O)
- Não pode jogar em casa ocupada
- Detecta vitória e empate

> Após rodar, tire um print e envie no tópico do projeto para liberar o próximo módulo.
